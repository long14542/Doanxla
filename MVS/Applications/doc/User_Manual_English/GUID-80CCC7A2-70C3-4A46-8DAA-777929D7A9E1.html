Main Window Introduction

After installation, you can run the Software.

The main window of the Software is shown as follows:

Figure 1 Main Window

The following table shows the description of each part of the main window.

Table 1 Main Window Description

No.

Area Name

Description

1

Menu Bar

You can access functions modules including File, View, Settings, Tool, and Help.

2

Control Toolbar

You can control file, set display window division, view acquisition, and perform other related operations.

3

Device List Panel

  • You can view all enumerated cameras and frame grabbers, including GigE Vision cameras, USB3 Vision cameras, Serial Port cameras, and cameras connected via GenTL and PCIe.

  • You can connect and disconnect, start and stop acquisition, and refresh enumerated devices.

4

Interface and Device Information Panel

You can view the information of selected frame grabber or camera from here.

5

Display Window

You can view the live view of selected camera, or view the image and video from local.

6

Feature Panel

You can view and configure features of selected camera, and perform other operations, including importing and exporting feature settings, and controlling User Set.
