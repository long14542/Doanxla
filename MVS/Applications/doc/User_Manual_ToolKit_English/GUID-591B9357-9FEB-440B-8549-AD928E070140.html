Port Editing Tool

The Port Editing Tool allows you to customize the frame grabber's port. For now, the tool only supports the Camera Link camera.

Note

Only user accounts with full control of the PC can modify the port.

Tool Introduction

The main page of the tool is shown below, and you can refer to the table for the related description.

Figure 1 Main Page of Tool
Table 1 Main Window Description

No.

Function

Description

1

Help

Select the language (Chinese and English) of the tool, and check the version information of the tool.

2

Port Configuration

Display the port of the current device and edit the port if needed.

3

Device Information

Display the connected device(s) and the corresponding COM port(s).

Port Configuration

Note

Currently, the tool only supports configuring the port for the self-developed Serial Port frame grabber.

  1. In the connected device list, select the frame grabber to be edited and the COM will be displayed on Current COM.

    Note

    You can click Update to update the COM information of devices.

  2. Click in the lower right of Virtual Port to select a COM port from the drop-down list.

  3. Click Apply.
