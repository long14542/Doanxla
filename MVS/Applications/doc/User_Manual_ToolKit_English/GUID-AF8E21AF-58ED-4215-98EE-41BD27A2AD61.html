Device Diagnostic Tool

Device Diagnostic Tool can monitor the running status of devices in real time, and display the event overview, event chart, IO waveform, and log records.

The Tool supports the following types of frame grabbers:

  • Camera Link frame grabber

  • Coax Press frame grabber

  • GigE frame grabber

  • 10 GigE frame grabber (support some models)

  • XoFLink Fiber Optic frame grabber

How To Diagnose

You can follow these steps to monitor the events and exceptions of specific devices:

  1. Open the Tool.

  2. In the Select Device window, select required devices.

  3. Click Run to start monitoring.
